 2. Tahun Kabisat
 Algoritma:
  -Tentukan ini Tahun berapa
  -Jika tahun habis di bagi 4, dan tidak habis dibagi 100 maka tahun Kabisat
  -Jika tahun habis di bagi 4, habis dibagi 100 dan habis dibagi 400 maka tahun Kabisat
  -Jika tidak maka bukan tahun Kabisat

  Pseudocode:
  SET tahun ini
  IF tahun ini % 4 EQUAL 0 
    AND  % 100 NOT EQUAL 0
  PRINT 'Tahun Kabisat'
  ELSEIF tahun ini % 4 EQUAL 0
    AND % 100 EQUAL 0
    AND % 400 EQUAL 0
  PRINT 'Tahun Kabisat'
  ELSE
  PRINT 'Bukan Tahun Kabisat'
  ENDIF

3. Laundry Day 
  Algoritma:
  - Masukkan Baju ke Mesin Cuci
  - Hitung Jumlah baju 
  - Jika  Jumlah baju belum 20, Maka masukkan baju lagi. 
  - Jika Jumlah baju sudah 20, Maka mulai mesin cuci. 

  Pseudocode:
  SET jumlah baju TO 0
  REPEAT
    READ jumlah baju 
    IF jumlah baju <20 
        INCREMENT jumlah baju
    ENDIF
  UNTIL jumlah baju = 20
  PRINT 'Mulai Mesin'

4. Periksa Kuku
   Algoritma:
   - Hitung jumlah siswa
   - Hitung panjang kuku 
   - Jika kuku panjang maka hukum siswa
   - Jika kuku pendek maka puji siswa
   - jika jumlah siswa sudah 40, maka selesai.
   
   Pseudocode:
   SET siswa to 1 
   SET kuku
   REPEAT
    IF kuku = Pendek 
     THEN PRINT 'Puji Siswa'
    ELSE 
     PRINT 'Hukum Siswa' 
    ENDIF
     INCREMENT siswa
   UNTIL siswa=40
